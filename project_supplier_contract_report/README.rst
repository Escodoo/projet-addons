Usage
=====

1. Acesse Configuracao > Contratos > Supplier Contract Templates.
2. Cadastre ou ajuste os templates de impressao.
3. Abra um contrato de fornecedor.
4. Clique em Print Contract ao lado de Criar Faturas.
5. Escolha o template e confirme em Print para gerar o PDF.
